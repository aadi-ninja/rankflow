===================================================
HOW TO USE RANKFLOW
===================================================

Thanks for downloading RankFlow! This tool lets you scrape TikTok ideas 
and mass-download video clips completely for free, using your own internet 
so you bypass all API limits and IP blocks.

REQUIREMENTS:
1. You must have Node.js installed on your computer.
   Download it here: https://nodejs.org/ (Get the "LTS" version)

HOW TO START (WINDOWS):
1. Double click the file named "Start-RankFlow.bat"
2. Wait a minute for it to install everything.
3. Your browser will automatically open to http://localhost:3000
4. Keep the black terminal window open while you use the app!

HOW TO START (MAC / LINUX):
1. Open your Terminal.
2. Type: cd /path/to/RankFlow/web
3. Type: npm install
4. Type: npm run dev
5. Open your browser to http://localhost:3000

TROUBLESHOOTING:
- "npm is not recognized": You did not install Node.js. Install it and restart your PC.
- Video downloads fail: Make sure you have python installed. YouTube/Instagram 
  occasionally block downloads if you do too many at once. Wait a few hours.

Enjoy!
